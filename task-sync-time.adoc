= Sincronizar o horário do sistema Cloud Volumes ONTAP usando o servidor NTP
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Para garantir a sincronização precisa do tempo, você deve configurar um servidor NTP (Network Time Protocol) para seus sistemas Cloud Volumes ONTAP . Certifique-se de configurar um servidor NTP para seus sistemas Cloud Volumes ONTAP em todos os provedores de nuvem para manter a sincronização de tempo consistente em sua rede.


NOTE: Se você não configurar um servidor NTP, poderá sofrer interrupções no serviço e sincronização de tempo imprecisa.

Você pode especificar um servidor NTP usando:

* https://docs.netapp.com/us-en/bluexp-automation/cm/api_ref_resources.html["API do NetApp Console"^].
* O comando ONTAP CLI https://docs.netapp.com/us-en/ontap-cli//cluster-time-service-ntp-server-create.html["criar servidor ntp de serviço de tempo em cluster"^].
